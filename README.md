# Hibernate PostgreSQL Project

## 📌 Overview
This project demonstrates how to use Hibernate ORM with PostgreSQL to manage database persistence in Java. It covers entity mapping, session management, and CRUD operations.

## ⚙️ Features
- Hibernate ORM integration
- PostgreSQL database connectivity
- Entity mapping with annotations
- CRUD operations (Create, Read, Update, Delete)
- Transaction management
- Example queries using HQL

## 🛠️ Requirements
- JDK 17+
- PostgreSQL installed locally
- Hibernate 5+ libraries
- PostgreSQL JDBC driver
